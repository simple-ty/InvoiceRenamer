# InvoiceRenamer 发票批量重命名工具

> 从一堆 PDF 发票中自动提取关键字段，按你定义的规则批量重命名，支持撤销与 Excel 导出。

## 功能特性

- **智能识别**：支持增值税专票/普票、电子发票、通行费发票等多种票面格式，自动提取开票日期、发票类型、销售方/购买方、金额等字段
- **非发票检测**：自动跳过非发票 PDF（如汇总单、明细单），避免误处理
- **自定义命名规则**：拖拽排列字段顺序，勾选启用字段，实时预览生成文件名
- **批量重命名 + 一键撤销**：确认后批量执行，操作有误可一键回滚
- **表头排序**：点击表头按任意列升序/降序排列
- **Excel 导出**：识别结果一键导出为 .xlsx，方便存档核对
- **单文件 exe**：32.8 MB，无需安装 Python 环境直接运行

## 快速开始

### 方式一：下载 exe（推荐）

1. 前往 [Releases](https://github.com/simple-ty/InvoiceRenamer/releases) 下载最新版 `InvoiceRenamer_vX.X.X.exe`
2. 双击运行（首次运行如被 Windows SmartScreen 拦截，点击「仍要运行」）
3. 选择发票所在文件夹 → 扫描 → 确认 → 重命名

### 方式二：源码运行

```bash
# 克隆仓库
git clone https://github.com/simple-ty/InvoiceRenamer.git
cd InvoiceRenamer

# 安装依赖
pip install -r requirements.txt

# 运行
python invoice_renamer_ui.py
```

**依赖清单**（仅 3 个第三方库，追求轻量）：

| 库 | 用途 |
|---|---|
| pdfplumber | PDF 文本提取与发票字段解析 |
| customtkinter | 现代化桌面 UI 框架 |
| openpyxl | Excel 导出 |

## 配置说明

### rules.json — 正则规则

发票字段提取的正则表达式集中存放在 `rules.json`，无需改代码即可调整识别规则：

```json
{
  "date": [...],           // 开票日期
  "number": [...],         // 发票号码
  "type_keywords": [...],  // 发票类型关键词
  "buyer_patterns": [...], // 购买方
  "seller_patterns": [...],// 销售方
  "amount_patterns": [...] // 金额
}
```

销售方/购买方名称自动截断到第一个主后缀，降低文件名长度：

|---|---|
| 中国石化销售股份有限公司湖南石油分公司 | 中国石化销售股份有限公司 |
| 湖南投资集团股份有限公司绕城公路西南段分公司 | 湖南投资集团股份有限公司 |

## 开发说明

### 打包 exe

```bash
# 确保 PyInstaller 已安装
pip install pyinstaller

# 一键打包
build.bat
```

输出：`dist/InvoiceRenamer_v{版本号}.exe`（约 32.8 MB）

打包配置说明：
- onefile 模式，单文件分发
- 无控制台窗口（windowed）
- 内嵌 rules.json + icon.ico
- 未启用 UPX 压缩（避免杀软误报）

## 技术特点

- **轻量依赖**：仅 3 个第三方库，exe 32.8 MB（曾尝试引入 PyMuPDF 加速，因体积 +37MB 移除）
- **保守识别策略**：非发票文件宁可跳过不误处理，三道关卡判定（强关键词 → 反关键词 → 核心字段+图片）
- **线程安全**：扫描在后台线程执行，通过 `root.after(0, ...)` 回主线程更新 UI
- **数据安全**：原始文件不动，改名前可预览，执行后可撤销
